# Engenharia de Dados — Estudos

Este repositório reúne **anotações, exercícios e práticas** desenvolvidos durante meus estudos em **Engenharia de Dados**.

Sou estudante de **Análise e Desenvolvimento de Sistemas**, em fase final da graduação, e gostei bastante da área de **dados**. Tenho interesse em entender como os dados são coletados, organizados, transformados e analisados, sempre buscando aplicar os conceitos na prática.

O foco deste repositório é:

* Consolidar os **fundamentos de dados**
* Praticar **Python aplicado a dados**
* Trabalhar com **SQL** e análise básica
* Registrar a evolução do aprendizado de forma clara e organizada

Este material representa um **processo real de aprendizado**, com melhorias contínuas conforme os estudos avançam.
